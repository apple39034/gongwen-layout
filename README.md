# gongwen-layout

中文公文排版格式规范参考及自动排版工具。

本项目包含两部分：

- **SKILL.md**：公文排版格式规范（字体、字号、页边距、标题层级、落款等）
- **scripts/gongwen_layout_v2.py**：Python 自动排版脚本，输入任意 docx，输出符合公文格式规范的 docx

---

## 排版规范速查

| 要素 | 格式 |
|------|------|
| 文件标题 | 方正小标宋简体 · 二号 · 居中 |
| 正文 | 仿宋三号 · 固定 28.95pt · 首行缩进 2 字符 · 两端对齐 |
| 一级标题（一、二、三……） | 黑体三号 · 段前段后 0 磅 · 左缩进 2 字符 |
| 二级标题 | 楷体三号 · 左缩进 2 字符 |
| 参考文献条目 | 仿宋三号 · 首行缩进 2 字符 · 两端对齐 |
| 落款 | 仿宋三号 · 右对齐 · 向右缩进 4 字符 |
| 页边距 | 上/下 36mm，左/右 27mm |
| 页眉/页脚 | 15mm / 27mm |
| 页码 | 底端外侧 · 宋体四号 |

### 标题层级示例

```
一、一级标题（黑体三号）
  （一）二级标题（楷体三号）
    1. 三级标题（仿宋三号）
      （1）四级标题（仿宋三号）
```

## 安装

```bash
pip install lxml
```

> 仅需 `lxml` 依赖，Python 标准库及 `python-docx`（内置）即可运行。

## 使用方法

```bash
python3 scripts/gongwen_layout_v2.py <输入文件.docx> [输出文件.docx]
```

- **输入文件**：原始 docx（内容不限，脚本会按公文结构重新生成正文）
- **输出文件**（可选）：默认覆盖输入文件

### 示例

```bash
# 覆盖原文件
python3 scripts/gongwen_layout_v2.py 我的公文.docx

# 输出到新文件
python3 scripts/gongwen_layout_v2.py 我的公文.docx 排版后.docx
```

---

## 注意事项

- 脚本会**覆盖**原始 docx 中的所有正文内容，按固定结构重建，使用前建议备份原文件
- 如需调整内容结构，请直接修改脚本中 `process()` 函数里的正文构建代码
- 统计学符号（*P*、*RR*、*OR*、*HR*、*CI*、*α*）会自动设置为斜体
- 表格自动按学术三线表格式输出（顶线/底线 1.5pt，分隔线 0.75pt，无填充色）
